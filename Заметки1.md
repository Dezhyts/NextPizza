В новой структуре app можно создавать вложенные layout’ы:

/app
  layout.tsx         // главный общий layout
  page.tsx           // главная страница
  /dashboard
    layout.tsx       // layout только для dashboard
    page.tsx         // страница dashboard
  /about
    page.tsx         // страница about использует общий layout


Главный layout.tsx будет оборачивать все страницы по умолчанию.
Внутренний dashboard/layout.tsx будет переопределять layout для страниц внутри /dashboard.

React.memo
Что делает: мемоизирует компонент, чтобы он не перерендеривался, если его пропсы не изменились.
useMemo
Что делает: кэширует результат вычисления между рендерами, чтобы не выполнять тяжёлую функцию заново.
На что влияет: не влияет на ререндер компонента сам по себе.
Дочерний компонент рендерится часто, а пропсы почти не меняются
Например, у тебя есть таблица или список с сотнями элементов, и родитель часто перерендеривается.
